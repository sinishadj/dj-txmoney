Package for working with money and currencies with automatic rate updates.

Documentation
-------------

The full documentation is at https://dj-txmoney.readthedocs.org.

Quickstart
----------

Install dj-txmoney::

    pip install dj-txmoney

Then use it in a project::

    import txmoney
